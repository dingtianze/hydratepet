## 🟡 一般风险

### R1: PWA推送通知兼容性
| 字段 | 内容 |
|------|------|
| **风险描述** | 不同浏览器和设备对PWA推送支持不一致，可能导致提醒功能在部分用户设备上失败 |
| **发生概率** | 中 (30-50%) |
| **影响程度** | 中 (核心功能受损) |
| **监测指标** | 推送到达率、用户反馈 |
| **应对措施** | 1. 多浏览器测试覆盖<br>2. 提供备用提醒方式（页面内动画）<br>3. 用户教育材料（如何开启推送权限） |
| **负责人** | senior-qa-engineer |
| **状态** | 🟡 监控中 |
| **计划关闭日期** | 发布前一周 |
